# Workout Tracker

A simple, effective workout tracking app for progressive strength training.

## Features
- 4-day Upper/Lower split with bonus Zone 5 cardio
- Auto-populated weights from previous sessions
- Rest timer with preset options
- Workout history and progress tracking
- Personal record (PR) tracking
- Week streak counter

## Program Structure
- **Upper 1:** Horizontal Push/Pull (DB bench, rows, accessories)
- **Lower 1:** Quad/Hip Dominant (goblet squat, Bulgarian split squats, leg press)
- **Upper 2:** Vertical Push/Pull (shoulder press, lat pulldowns, arms)
- **Lower 2:** Posterior/Unilateral (RDLs, lunges, hamstring work)
- **Bonus:** Zone 5 cardio options (optional 5th day)

## Usage
Open the app in your browser. Add to home screen on mobile for app-like experience.

All data is stored locally in your browser.
